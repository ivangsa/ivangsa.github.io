# Ivan Garcia Sainz-Aja - Personal Website

Event-Driven Architecture expert's personal website built with Astro.

## 🚀 Project Structure

This site showcases articles, talks, and projects focused on Event-Driven Architecture, AsyncAPI, Kafka ecosystems, and domain modeling.

## 🧞 Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |

## 🌐 Deployment

This site is deployed to GitHub Pages at [ivangsa.com](https://ivangsa.com) via the repository [ivangsa/ivangsa.github.io](https://github.com/ivangsa/ivangsa.github.io).

Deployment happens automatically on push to `main` branch.
